# 应用图标占位说明

请将以下图标文件放置在此目录：

## Windows
- icon.ico (256x256, 包含多尺寸: 16, 32, 48, 64, 128, 256)

## macOS (如需支持)
- icon.icns

## Linux (如需支持)
- icon.png (512x512)

---

## 图标设计建议

根据 Neon Tools 的霓虹风格，图标应包含：

1. **基础形状**：工具箱图标或齿轮
2. **霓虹配色**：
   - 主色：霓虹青色 #21e6ff
   - 辅色：霓虹粉色 #ff2aa1 或紫色 #9b5cff
3. **发光效果**：外发光/光晕效果
4. **厚描边**：2-3px 粗边框
5. **深色背景**：#0a0e27 或透明

## 临时方案

在未提供图标前，electron-builder 会使用默认图标。
应用功能不受影响。
